# adapters/ — model weights

The trained adapter weights (DoRA and full fine-tune variants) are not stored
in this repository — they are too large (>20 GB) for GitHub.

They will be deposited on Zenodo with their own DOI after the thesis defense.
See `../../../data/README.md` for the link once minted.

To run the pipeline without downloading the weights, see the demo notebook
in `chapter2-ner/evaluation/` which uses smaller baseline models.
