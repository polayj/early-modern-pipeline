# corpus-references/

Citation manifest for the 20,000+ documents used in Chapter 3's GraphRAG system.

## Status: TO BE POPULATED IN PHASE B

The citation manifest lives on the school computer alongside the OCR'd document store. It will be added here in Phase B with the following files:

- `archive-org-items.csv` — one row per Internet Archive document (item ID, title, author, year, URL)
- `eebo-items.csv` — one row per EEBO document (STC or Wing number, title, author, year)
- `customs-ledgers.csv` — provenance for the 57 import-records files in `../customs-ledgers/`
- `selection-criteria.md` — how the 20K corpus was assembled

### Copyright stance

- Internet Archive items: full citation + URL; OCR text deposited separately on Zenodo
- EEBO items: STC/Wing number + author/title/year only; OCR text **withheld** pending librarian review (see top-level README)

See `../../docs/PHASE-B-HANDOFF.md` for build instructions.
